### “信息素养”是一种基本能力，是一种对信息社会的适应能力。美国教育技术CEO论坛2001年第4季度报告提出21世纪的能力素质，包括基本学习技能(指读、写、算)、信息素养、创新思维能力、人际交往与合作精神、实践能力。信息素养是其中一个方面，它涉及信息的意识、信息的能力和信息的应用。
